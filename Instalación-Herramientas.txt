--PHP ARTISAN

Servidor de desarrollo
php artisan serve: Inicia el servidor de desarrollo local en http://localhost:8000.
php artisan serve --port=8080: Cambia el puerto por defecto.
php artisan serve --host=0.0.0.0: Permite acceso desde otras máquinas en la red (útil con Vagrant). 
Migraciones y base de datos
php artisan make:migration create_users_table: Crea una nueva migración.
php artisan migrate: Ejecuta las migraciones pendientes.
php artisan migrate:reset: Deshace todas las migraciones.
php artisan migrate:refresh: Vuelve a ejecutar todas las migraciones.
php artisan migrate:fresh --seed: Elimina todas las tablas y las recrea, luego ejecuta los seeders. 
Modelos, controladores y recursos
php artisan make:model User: Crea un modelo.
php artisan make:model User -mcr: Crea modelo, migración, controlador y factory.
php artisan make:controller UserController --resource: Crea un controlador con métodos CRUD.
php artisan make:controller UserController --api: Crea un controlador para API REST (sin métodos create y edit).
php artisan make:controller UserController --model=User: Crea un controlador vinculado a un modelo existente. 
Seeders y factories
php artisan make:seeder UsersSeeder: Crea un seeder.
php artisan db:seed: Ejecuta todos los seeders.
php artisan db:seed --class=UsersSeeder: Ejecuta un seeder específico. 
Caché y optimización
php artisan config:cache: Almacena la configuración en caché (producción).
php artisan route:cache: Almacena las rutas en caché.
php artisan view:cache: Compila las vistas en caché.
php artisan cache:clear: Limpia la caché de la aplicación.
php artisan config:clear: Limpia la caché de configuración.
php artisan route:clear: Limpia la caché de rutas.
php artisan optimize: Optimiza la aplicación para producción. 
Otros comandos útiles
php artisan route:list: Muestra todas las rutas definidas.
php artisan tinker: Abre una consola interactiva para probar código PHP.
php artisan make:component Alert: Crea un componente Blade personalizado.
php artisan make:request StoreUserRequest: Crea una clase de validación para requests.
php artisan list: Muestra todos los comandos disponibles. 
💡 Consejo: Usa php artisan help [comando] para ver detalles específicos de cualquier comando. 

---COMANDOS LIVEWIRE

-- php artisan make:livewire NombreComponente
Crea un nuevo componente Livewire con una clase PHP y una vista Blade asociada.

-- php artisan make:livewire NombreComponente --inline
Crea un componente inline, donde la vista (Blade) se incluye directamente en el método render() del archivo PHP. 

php artisan livewire:publish --assets
Publica los assets (CSS, JS) de Livewire para que estén disponibles en tu aplicación. 

php artisan livewire:publish --config
Publica el archivo de configuración de Livewire. 

php artisan livewire:stub
Personaliza los stubs (plantillas) que se usan al crear nuevos componentes. 

php artisan livewire:clear
Limpia la caché de componentes Livewire. 

----Uso de directivas en vistas Blade

-- wire:model
Sincroniza un campo de entrada con una propiedad del componente (actualiza al perder el foco o al enviar el formulario). 

-- wire:model.live
Sincroniza en tiempo real (sin necesidad de perder el foco). 

-- wire:click
Asigna una acción al hacer clic en un elemento. 

-- wire:submit
Maneja el envío de formularios.

@livewireStyles y @livewireScripts
Incluyen los estilos y scripts necesarios de Livewire en plantillas Blade. 

--Ejemplo de uso
php artisan make:livewire Tareas

Este comando genera:

app/Http/Livewire/Tareas.php (clase PHP)
resources/views/livewire/tareas.blade.php (vista)
Para usarlo en una vista:

@livewire('tareas')
